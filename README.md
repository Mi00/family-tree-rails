Application shows relations between users.
You can download all or one user as pdf or csv.

Sample admin user:
	email: bbbb@wp.pl
	pass: 111111

Sample normal user:
	email: bbb@wp.pl
	pass: 111111

This app is available online:
https://family-tree-app-by-michal.herokuapp.com/

Ruby ver. 2.4.2

Before new run you can add some data:

rails db:seed
